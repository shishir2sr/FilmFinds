## API Reference

#### Get all items

```http
  GET https://api.themoviedb.org/3/search/movie
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. My API key |

#### Get item

```http
  GET https://api.themoviedb.org/3/search/movie/discover/movie
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. My API key |
